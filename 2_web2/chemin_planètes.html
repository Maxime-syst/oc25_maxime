Santier des planètes


Tracé de SuisseMobile

Informations :

  • Saint-Luc
  • distance: 6.2km
  • 343 m dénivelé positif
  • 141 m dénivelé négatif
  • durée: 1h 49min

Description :

Pour cette randonnée familliale, rendez vous dans le village de St-Luc, dans le Val d'Anniviers. Une fois arrivé, il vous faudra prendre les remontées mécaniques, qui relient le village à Tignousa. Depuis là, vous pourrez empruntez le santier des planètes, qui est relativement plat et adapté aux petits comme aux grands. Vous partirez en direction de l'hotel Weisshorn, en empruntant le chemin du mythique parcours de Sierre-Zinal. Une fois à l'hotel vous avez l'option de cotinuer le santier ou vous pouvez directement redescendre sur le vllage de St-Luc, ce qui vous permet de faire une boucle.
